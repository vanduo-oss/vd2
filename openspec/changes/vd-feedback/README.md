# vd-feedback

P0-3 of the `full-capability-transfer` plan. Adds 3 Vd* components
(`VdChip`, `VdSkeleton`, `VdPreloader`) and 3 component pages. No
framework repo work — all components are Vue-native reimplementations
using existing framework CSS selectors.

See `proposal.md`, `design.md`, `tasks.md`, and
`specs/feedback/spec.md`.
