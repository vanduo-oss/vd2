# vd-data-display

P0-2 of the `full-capability-transfer` plan. Adds 4 Vd* components
(`VdAvatar`, `VdTable`, `VdCollection`, `VdBreadcrumb` — the last
already existed in vd2) and 4 component pages. No framework repo
work — all components are Vue-native reimplementations using existing
framework CSS selectors (`vd-avatar-*`, `vd-table-*`, `vd-collection-*`,
`vd-breadcrumb-*`).

See `proposal.md`, `design.md`, `tasks.md`, and
`specs/data-display/spec.md`.
