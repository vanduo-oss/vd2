# vd-effects

P1-2 of the `full-capability-transfer` plan. Adds 3 explanation
pages under `/effects/*` (glass, morph, parallax). No new Vd*
components — the effects pages demonstrate the framework's
`vd-glass*`, `vd-morph*`, and `vd-parallax*` CSS modules.

See `proposal.md`, `design.md`, `tasks.md`, and
`specs/effects/spec.md`.
