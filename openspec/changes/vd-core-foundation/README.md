# vd-core-foundation

P1-1 of the `full-capability-transfer` plan. Adds 6 explanation
pages under `/core/*` (color palette, typography, icons, golden
ratio, grid system, shadows & glow). No new Vd* components — the
foundation pages are long-form content that uses existing framework
classes and tokens.

See `proposal.md`, `design.md`, `tasks.md`, and
`specs/core-foundation/spec.md`.
